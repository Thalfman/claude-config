# Action Verbs

Strong action verbs by category. Pick the one that most precisely describes what you actually did. **Don't reach for a "stronger" verb if it would distort the truth** — "led" is a fabrication if you contributed; "contributed to" is fine.

## Leadership / People

| Tier | Verbs |
|---|---|
| Owned outcome | led, directed, headed, owned, drove, spearheaded, ran |
| Coordinated | coordinated, organized, orchestrated, aligned, facilitated |
| Coached | mentored, coached, guided, trained, advised, onboarded |
| Hired / staffed | hired, recruited, staffed, interviewed, scaled (a team) |
| Influenced | influenced, persuaded, championed, advocated, secured buy-in |

**Calibration warning:** "led" implies you owned the outcome (success or failure was on you). If two people co-led, say "co-led". If you were a senior IC who shaped the work without ownership, "drove" or "championed" is more honest than "led".

## Build / Create / Ship

| Tier | Verbs |
|---|---|
| From scratch | built, created, developed, designed, architected, founded, launched, shipped, delivered |
| Composed | assembled, integrated, composed, wired, connected, productionized |
| Prototyped | prototyped, piloted, experimented, validated |

**Calibration warning:** "Architected" should be reserved for actual architecture work (system design, trade-off analysis, RFC authorship). For "wrote a service", say "built" or "developed".

## Improve / Optimize

| Tier | Verbs |
|---|---|
| Increased | increased, grew, expanded, accelerated, scaled |
| Decreased | reduced, cut, trimmed, eliminated, consolidated, streamlined |
| Enhanced | improved, enhanced, upgraded, refactored, modernized, hardened |
| Stabilized | stabilized, restored, recovered, debugged, fixed, resolved |

**Pair improvement verbs with metrics whenever the master has them.** "Reduced p99 latency by 35%" beats "Reduced p99 latency".

## Verbs to avoid

These signal weak ownership or vague work. Replace.

| Weak | Try |
|---|---|
| "Worked on" | built, designed, developed, contributed to |
| "Helped with" | supported, partnered with, contributed to (be specific about what you did) |
| "Assisted in" | partnered, contributed, supported (or just lead with your specific action) |
| "Responsible for" | (drop and use the actual action verb — "owned", "ran", "managed") |
| "Was tasked with" | (drop and use the actual action verb) |
| "Involved in" | (drop and use the actual action verb) |
| "Familiar with" | used, applied, deployed (only if true; otherwise leave out) |
| "Exposure to" | (do not use — implies no real experience) |
| "Successfully" | (drop — outcome should be in the metric) |
| "Utilized" | used (shorter is better) |
| "Various" | (drop — be specific) |

## Tense

- Past tense for past roles (built, led, designed)
- Present tense for current role (build, lead, design)
- Be consistent within each role
